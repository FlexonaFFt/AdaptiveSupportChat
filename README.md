# AdaptiveSupport

AdaptiveSupport — это проект про «умную первую линию поддержки»: пользователь пишет в Telegram, а бот быстро разбирает вопрос и старается закрыть его сразу, без ожидания оператора. Если тема нестандартная или уверенность низкая, бот не выдумывает ответ, а аккуратно переводит диалог на живого специалиста.

Фишка в том, что ответы строятся не «с потолка», а на основе вашей базы знаний через RAG, плюс это все легко разворачивается как обычный API-сервис. Внутри модули разделены по ролям (`supportbot`, `api`, `mlcore`, `core`), поэтому проект удобно развивать: отдельно прокачивать качество ответов, отдельно инфраструктуру и отдельно сценарии поддержки.
